# RumiCar-C3

## 車体

3DプリンターはPrusa MK4、フィラメントはPLAフィラメントを使用しています。  
[部品表](./部品表.xlsx)  
本で見つけたアッカーマン・ジャントー方式を取り入れてみました。  
サーボモーターでステアリングを操作できます。
